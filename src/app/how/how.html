with {{id}}

Before installing any plugins, you need to install the CLI. But before you can install the CLI, you will need Node.js. You can find instructions for installing Node on nodejs.org.

Now that you have Node installed, you can install the Minecraft CLI by running the following:

$ npm install -g minecraft-cli

More details on the Minecraft CLI can be found here.


Installing {{id}}
$ mc install {{id}}

With the Minecraft CLI, you can now install your plugin by running the above. You will notice a plugins directory apear in the current path.
